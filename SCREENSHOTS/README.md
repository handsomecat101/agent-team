# Thư mục Screenshots

> Lưu trữ hình ảnh lỗi khi agent tìm thấy bằng trình duyệt

---

## 📂 Cấu trúc:

```
SCREENSHOTS/
├── 2026-03-14/
│   ├── bug-login-001.png
│   └── ...
├── 2026-03-15/
│   └── ...
└── README.md
```

---

## 📝 Cách sử dụng:

1. **Tạo folder ngày** nếu chưa có: `SCREENSHOTS/2026-03-14/`
2. **Lưu screenshot** với tên mô tả: `bug-[ten-loi]-[ID].png`
3. **Ghi đường dẫn** vào ISSUE_LOG.md hoặc AGENT_CHAT.md

---

## 📋 Đặt tên file:

```
bug-[ten-loi]-[ID].png

Ví dụ:
- bug-login-redirect-001.png
- bug-button-color-002.png
- bug-dark-mode-003.png
```

---

*Folder này dùng chung cho team - agents save screenshots vào đây*
