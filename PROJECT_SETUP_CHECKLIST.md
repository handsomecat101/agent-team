## SECTION 1: PROJECT IDENTITY

### Câu hỏi cơ bản

```
1. Tên project là gì?
   → [Trả lời]

2. Project này làm gì? (Mô tả ngắn 1-2 câu)
   → [Trả lời]

3. Project type là gì?
   [ ] Web App
   [ ] Mobile App
   [ ] API
   [ ] Tool/Utility
   [ ] Khác: __________

4. Tech stack sử dụng?
   [ ] HTML/CSS/JS
   [ ] React/Vue/Angular

   [ ] Khác: __________
```

## 📝 SAU KHI TRẢ LỜI

Sau khi trả lời xong tất cả câu hỏi:

1. **Copy câu trả lời** vào SOUL.md
2. **Copy câu trả lời** vào PROJECT_STATE.md (phần thành viên)
3. **Thêm constraints** vào GOVERNANCE.md (nếu cần)
4. **Bắt đầu** project!
